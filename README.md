# Calculator Module with Pytest

## Description
A simple Python calculator module developed to perform basic arithmetic operations.
Unit tests are written using Pytest to validate correctness of the functions.

## Features
- Addition, subtraction, multiplication, and division
- Unit tests for each operation
- Handles basic edge cases

## Technologies Used
- Python
- Pytest

## Learning Outcomes
- Understanding modular Python code
- Writing and running unit tests using Pytest
- Importance of testing in software development
